## Prerequisites

<Callout type="warn" title="Before You Start">
Make sure your Pterodactyl Panel is up-to-date or atleast above `v1.10.x`.
</Callout>

## Installation

<Steps>

<Step>
### Upload Extension Files

Upload the contents inside the `uploadme` folder included with the extension into your Pterodactyl directory.

<Callout type="info">
The default Pterodactyl directory is typically located at `/var/www/pterodactyl`
</Callout>

</Step>

<Step>
### Register Frontend Routes

Open the file: `resources/scripts/routers/routes.ts`

**Step 1:** Find the following import statement:

```ts title="resources/scripts/routers/routes.ts"
import ServerActivityLogContainer from "@/components/server/ServerActivityLogContainer";
```

**Step 2:** Add the MC Mods import **directly below it**:

```ts title="resources/scripts/routers/routes.ts" {2}
import ServerActivityLogContainer from "@/components/server/ServerActivityLogContainer";
import MCModsRoutes from "@/routers/mcmods";
```

**Step 3:** Find the file edit route:

```js title="resources/scripts/routers/routes.ts"
{
    path: '/files/:action(edit|new)',
    permission: 'file.*',
    name: undefined,
    component: FileEditContainer,
},
```

**Step 4:** Add the MC Mods routes **directly below it**:

```js title="resources/scripts/routers/routes.ts" {6}
{
    path: '/files/:action(edit|new)',
    permission: 'file.*',
    name: undefined,
    component: FileEditContainer,
},
...MCModsRoutes,
```

![Frontend Routes Configuration](/docs/pterodactyl/extensions/mcmods/help-1.png)

</Step>

<Step>
### Register API Routes

Open the file: `routes/api-client.php`

Add the following code **at the end of the file**:

```php title="routes/api-client.php"
Route::prefix('/extensions/mcmods')->group(
    base_path('routes/mcmods/api-client.php')
);
```

![API Routes Configuration](/docs/pterodactyl/extensions/mcmods/help-2.png)

</Step>

<Step>
### Register Admin Routes

Open the file: `routes/admin.php`

Add the following code **at the end of the file**:

```php title="routes/admin.php"
include 'mcmods/admin.php';
```

![Admin Routes Configuration](/docs/pterodactyl/extensions/mcmods/help-3.png)

</Step>

<Step>
### Add Admin Navigation

Open the file: `resources/views/layouts/admin.blade.php`

**Step 1:** Find the Nests navigation block:

```php title="resources/views/layouts/admin.blade.php"
<li class="{{ ! starts_with(Route::currentRouteName(), 'admin.nests') ?: 'active' }}">
    <a href="{{ route('admin.nests') }}">
        <i class="fa fa-th-large"></i> <span>Nests</span>
    </a>
</li>
```

**Step 2:** Add the MC Mods navigation **directly below it**:

```php title="resources/views/layouts/admin.blade.php" {6-10}
<li class="{{ ! starts_with(Route::currentRouteName(), 'admin.nests') ?: 'active' }}">
    <a href="{{ route('admin.nests') }}">
        <i class="fa fa-th-large"></i> <span>Nests</span>
    </a>
</li>
<li class="{{ ! starts_with(Route::currentRouteName(), 'admin.mcmods.index') ?: 'active' }}">
    <a href="{{ route('admin.mcmods.index') }}">
        <i class="fa fa-cubes"></i> <span>MC Mods</span>
    </a>
</li>
```

![Admin Navigation Configuration](/docs/pterodactyl/extensions/mcmods/help-4.png)

</Step>

<Step>
### Clear Cache

Run the following command to clear the application cache:

```bash
php artisan optimize:clear
```

</Step>

<Step>
### Build Panel Assets

Rebuild the Pterodactyl panel assets to compile your changes.

Follow the [official Pterodactyl customization guide](https://pterodactyl.io/community/customization/panel.html#building-assets) for detailed instructions.

```bash
yarn
yarn build:production
```


</Step>

</Steps>

## Configuration

<Steps>

<Step>
### CurseForge API Key

MC Mods Manager uses the **CurseForge API** to provide access to a large library of Minecraft: Java Edition Mods.

1. Visit [CurseForge Console](https://console.curseforge.com)
2. Create an account or sign in
3. Navigate to the API Keys section
4. Generate a new API key
</Step>

<Step>
### Configure the Extension

Enter your CuseForge API key in the MC Mods Manager settings within the Pterodactyl admin panel.
</Step>

</Steps>

---

## Finished!

Your **MC Mods Manager** extension is now installed and ready to use.
