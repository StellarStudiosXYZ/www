# Installation

Upload the contents of the `pterodactyl` directory (included with this archive) to your Pterodactyl directory (usually located at `/var/www/pterodactyl`).

<Steps>

<Step>
  <div className="space-y-6">

### Frontend Routes

Open the file: `resources/scripts/routers/routes.ts`

Find the code:

```ts
import ServerActivityLogContainer from '@/components/server/ServerActivityLogContainer';
```

Add this code below it:

```ts
import MCPluginsRoutes from '@/routers/extensions/mcplugins';
```

Find this code:

```ts
{
    path: '/files',
    permission: 'file.*',
    name: 'Files',
    component: FileManagerContainer,
},
```

Add this code below it:

```ts
...MCPluginsRoutes,
```

![MC Plugins Help-1](/mcplugins/help-1.png)

  </div>
</Step>

<Step>
  <div className="space-y-6">

### Admin Sidebar

Open the file: `resources/views/layouts/admin.blade.php`

Find this code:

```html
<li class="{{ ! starts_with(Route::currentRouteName(), 'admin.nests') ?: 'active' }}">
    <a href="{{ route('admin.nests') }}">
        <i class="fa fa-th-large"></i> <span>Nests</span>
    </a>
</li>
```

Add this code below it:

```php
@include('layouts.extensions.mcplugins')
```

# Configure CurseForge

MC Plugins Manager uses CurseForge API to provide access to a large library of Minecraft plugins. To obtain your key follow the steps below:

<Steps>

<Step>
  <div className="space-y-6">
    Create a account at [CurseForge Console](https://console.curseforge.com)
  </div>
</Step>

<Step>
  <div className="space-y-6">
    Generate an API key and paste it in the extension's settings.
  </div>
</Step>

</Steps>

# Wings Modification

You only need to perform Wings modifications if you want to use the **Manage Plugins** feature (show installed plugins, update plugins, and remove plugins).
These steps must be performed on every node running Wings and make sure you don't run the commands below inside your Pterodactyl directory (usually located at `/var/www/pterodactyl`)

<Steps>

<Step>
  <div className="space-y-6">

### Install Go

Go is required to build Wings. This guide assumes you are on a Debian-based system. If you are not using Debian-based system, install Go using your system's package manager.

```bash
apt-get install -y golang
```

  </div>
</Step>

<Step>
  <div className="space-y-6">

### Clone Wings Repository

This will clone the Pterodactyl Wings repository.

```bash
git clone https://github.com/pterodactyl/wings.git && cd wings
```

  </div>
</Step>

<Step>
  <div className="space-y-6">

### Apply the Wings Patch

Upload the `wings.patch` file included with the archive into the Wings directory that you just cloned then apply the patch by running the command below:

```bash
git apply wings.patch
```

  </div>
</Step>

<Step>
  <div className="space-y-6">

### Build Wings Binary

This will generate the Wings binary under the `build/` directory.

```bash
make build
```

  </div>
</Step>

<Step>
  <div className="space-y-6">

### Replace the Wings Binary

This will stop Wings, replace the binary, and start Wings again.

```bash
systemctl stop wings
cp "build/wings_linux_$(uname -m | sed -e 's/x86_64/amd64/' -e 's/aarch64/arm64/')" /usr/local/bin/wings
systemctl start wings
```

  </div>
</Step>

</Steps>
